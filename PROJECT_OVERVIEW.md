## 3. Ключевые проблемы и решения

### Автотаргетинг = B2C трафик
- **Проблема:** 52% бюджета шло на автотаргетинг (5,505₽ за 3 дня в группе «Бронирование»), приводил только физлиц
- **Решение:** оставлены минимальные категории автотаргетинга (Альтернативные + Бренд), добавлены 108 минус-слов
- **Ограничение API:** автотаргетинг нельзя отключить через API (ошибка 5005), только вручную в интерфейсе

### BidCeiling = контроль CPC
- **Проблема:** без BidCeiling CPC достигал 62₽
- **Решение:** установлен BidCeiling — 60₽ для поиска, 15₽ для РСЯ

### Нишевые B2B ключи = нет трафика
- **Проблема:** после оптимизации трафик упал до 0 (слишком узкие ключи + низкий BidCeiling)
- **Решение:** добавлены 22 ключа с оператором `+` для расширения охвата, BidCeiling поднят до 60₽

### Ограничения Yandex Direct API v5
- `keywords.suspend` не работает для автотаргетинга (ошибка 5005)
- `keywords.delete` не работает для автотаргетинга (warning 5005)
- Endpoint `autotargetings` возвращает 404
- `TextAdGroupAutoTargetingCategories` — неизвестный параметр
- `KeywordBids.Bid` не существует для WB_MAXIMUM_CLICKS — нужен `SearchBid`
- `SearchBid` возвращает 0 результатов для WB_MAXIMUM_CLICKS (ставки управляются на уровне стратегии)
- `AdImageHash` — одно изображение на объявление (нужно создавать отдельные объявления для разных форматов)
